WELCOME
Well howdy there!

My name is Michael Anthony Francis Rubino. Mike or Michael on their own work just as well if you prefer. I've worn a number of different hats throughout my career, including graphic designer, video game artist, teacher, etc. Nowadays I wear the hat of a full-stack web developer.

While web development has become my main focus in life, there are still a number of things I love doing as hobbies for the saking of bettering myself. Chief among those hobbies are my creative interests such as digital painting, illustration, and writing. I've always had a creative spark inside of me, and I try to channel that creativity in whatever adventures I decide to set off on.

WHAT CAN I DO?

WEB DEVELOPMENT

I'm highly skilled with in HTML5, CSS3, Javascript, JQuery, MySQL, Node, Express, React, Redux, AJAX, as well as Git and Github.

WEB DESIGN

With my designs I tend to focus on stylish aesthetic working in tandem with a strong consideration for the user's perspective.

RESPONSIVE DESIGN

Given how much time spent browsing the web is spent on mobile devices, I've built up a tremendous toolkit for myself when it comes to responsive design.

GRAPHIC DESIGN

I have over 6 years of experience working in the graphic design industry, and those design sensibilites influence my sensibilites for web design.
